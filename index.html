Hopfield Network

This project aims to simulate a neural machine : Mind 1024 Therefore I coded a hopfield network, which is a weighted network with N neurons fully connected to each other. The network is used as an associative memory, it can retrieve a memorized pattern given a noisy input. In this project we are trying to use it for classification of digits.
